Electron的坑

#1.node install.js卡住

可能是网络原因,可以等待或者将npm的下载库改为百度

#2.发生错误 "Cannot find module app"

原代码长这样：

var app = require('app');
var BrowserWindow = require('browser-window');

修改成:

const electron = require('electron');
const app = electron.app;
const BrowserWindow = electron.BrowserWindow;

参考:https://stackoverflow.com/questions/39683163/cannot-find-module-app-when-trying-to-run-electron-app

#3.loadUrl报错

修改成loadURL() 参考:https://stackoverflow.com/questions/37614054/loadurl-is-not-working-in-electron
